Le Modèle d'Intelligence Artificielle

Découvrez comment ce modèle de deep learning a été conçu et entraîné pour détecter la malaria avec une précision de 94%.

Architecture du Modèle

Notre modèle utilise une architecture CNN (Convolutional Neural Network) optimisée pour la détection de parasites de la malaria dans les images microscopiques.

  • 3 couches de convolution avec BatchNormalization
  • MaxPooling pour la réduction de dimension
  • Dropout pour prévenir le surapprentissage
  • Dense layers avec activation ReLU 
model = tf.keras.Sequential([
    Conv2D(32, (3, 3), activation='relu'),
    BatchNormalization(),
    MaxPooling2D((2, 2)),
    Dropout(0.25),

    Conv2D(64, (3, 3), activation='relu'),
    BatchNormalization(),
    MaxPooling2D((2, 2)),
    Dropout(0.25),

    Conv2D(128, (3, 3), activation='relu'),
    BatchNormalization(),
    MaxPooling2D((2, 2)),
    Dropout(0.25),

    Dense(256, activation='relu'),
    BatchNormalization(),
    Dropout(0.5),
    Dense(2, activation='softmax')
])

Métriques de Performance

Sensibilité

93.8%

La sensibilité mesure la capacité du modèle à identifier correctement les cas positifs de malaria. Sur 100 images contenant réellement des parasites, le modèle en détecte correctement environ 94.

Formule : Vrais Positifs / (Vrais Positifs + Faux Négatifs)

Spécificité

94.2%

La spécificité mesure la capacité du modèle à identifier correctement les cas négatifs (absence de malaria). Sur 100 images sans parasite, le modèle en identifie correctement environ 94.

Formule : Vrais Négatifs / (Vrais Négatifs + Faux Positifs)

Pourquoi ces métriques sont importantes ?

  • Une haute sensibilité est cruciale pour ne pas manquer de vrais cas de malaria, ce qui pourrait avoir des conséquences graves pour les patients.
  • Une haute spécificité évite les faux diagnostics positifs, réduisant ainsi les traitements inutiles et l'anxiété des patients.

Dataset d'Entraînement

Le modèle a été entraîné sur un dataset équilibré de 27,558 images microscopiques de cellules sanguines.

Images parasitées 13,779 images
Images saines 13,779 images

Prétraitement des Images

Chaque image est prétraitée avant d'être analysée par le modèle pour garantir des résultats optimaux.

  • Redimensionnement à 128x128 pixels
  • Normalisation des valeurs de pixels
  • Augmentation de données (rotation, zoom)
